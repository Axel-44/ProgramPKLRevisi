## Instalasi

1. Clone repository ini
2. Install dependencies:
   \`\`\`bash
   npm install
   \`\`\`

3. Jalankan development server:
   \`\`\`bash
   npm run dev
   \`\`\`

4. Buka browser dan akses `http://localhost:3000`

## Build untuk Production

\`\`\`bash
npm run build
\`\`\`
